什么是 Coding Manager？

一个本地运行的项目管理工具，专为 vibe coding 创作者设计。
当你同时维护多个 AI 辅助项目时，Coding Manager 让你一眼掌握所有进度。

核心功能

📁
自动扫描项目
配置本地目录路径后，Coding Manager 自动发现所有 git 仓库，免手动录入。每次同步自动拉取最新 commit，再也不用逐个仓库去看。
🤖
AI 每日摘要
每天同步后，Claude 自动阅读 commit message 和 diff，生成一段自然语言摘要——不只说改了什么，还会指出「放弃了什么」。需要配置 Claude API Key。
✅
每日四项清单
追踪四个好习惯：更新了 CLAUDE.md、更新了 Memory、推送到 GitHub、完成部署。每天同步时自动检测前三项，部署状态手动打钩。
📝
笔记 & 灵感
给每个项目记开发笔记，也可以写与项目无关的学习笔记、收藏参考资料。笔记支持标签、关联到具体某一天的日志，一键导出 Markdown。
☑️
待办管理
全局待办和项目待办两个层级。项目待办标为「重要」后自动浮到顶部面板，配合每日清单状态一起看，让你每次打开就知道今天该做什么。
📣
自媒体内容管理
把内容创作的选题、状态、发布时间统一管理在这里，和对应的开发项目关联起来。从「规划中」到「待发布」到「已发布」，全程追踪。

如何开始

  1. 1
    克隆仓库，安装依赖
    git clone https://github.com/jasminemo1110/coding-manager.git
    cd coding-manager
    python3 -m venv .venv && source .venv/bin/activate
    pip install -r requirements.txt
  2. 2
    启动服务
    python app.py
    打开浏览器访问 http://localhost:8765
  3. 3
    填写配置（可选但推荐）
    进入「设置」页，填写 Claude API Key 以启用 AI 摘要；填写 GitHub Token 以自动识别私有仓库可见性。
  4. 4
    添加项目
    在设置页配置本地扫描目录，Coding Manager 会自动扫描所有 git 仓库，逐个确认添加或排除。
  5. 5
    每天工作完，点「同步今天」
    自动拉取 commit、生成 AI 摘要、检测清单三项状态，当天进展一目了然。
